Informações do Projeto

  • Data do Projeto: 29 de Abril, 2024
  • URL do Projeto: Dt-Money

Descrição

O dtmoney é um projeto desenvolvido em React, utilizando styled components e outras bibliotecas para construção de interfaces. Ele é basicamente um sistema de controle financeiro, permitindo ao usuário registrar suas despesas e receitas, e mostrando o saldo atual.

Funcionalidades

  • Registro de despesas e receitas
  • Calcula automaticamente o saldo atual
  • Visualização de histórico de transações

Tecnologias Utilizadas

  • React.js: Uma biblioteca JavaScript para construir interfaces de usuário.
  • Styled Components: Uma biblioteca para estilização de componentes em React.
  • Outras bibliotecas React para funcionalidades específicas, como controle de estado e roteamento.
